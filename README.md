# Video Annotation Portfolio

## Project Description
This repository demonstrates sample video and image annotation work used for training computer vision and AI models. The project includes object detection annotations, human activity recognition labels, and structured dataset preparation.

## Dataset Structure
traffic-video-annotation/
human-activity-annotation/
dataset-description/

## Annotation Format
Annotations are stored in JSON format and contain object labels and bounding box coordinates.

## Tools Used
- CVAT
- Labelbox
- Manual bounding box annotation
